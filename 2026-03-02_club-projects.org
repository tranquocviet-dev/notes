The workshop's goals is to provide knowledge to people about software cracking make them understand that cracking can help people with more legal activities
** What is software cracking
- Software cracking is the act of removing the copy protection of a software
- It used to be known as "breaking" in the 1980s
- It can mean many things
  - modifying code of the original program
  - sharing stolen keys to the program
  - generating keys to the software through a third party program
** Different methods of software cracking

** Practical usage of software cracking knowledge
