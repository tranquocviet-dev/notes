# Sub sets,... hmm... Subsets?
Sub sets,... no wait they are Subsets, are a collection of stuff from an existing set. Say, there is a set made out of three elements. Each one of those elements are a subset. Two random ones can create a subset. So, as long as you have any set, you can create a subset of it, really. They are illustrated as $subset.eq$.

Well, technically, a set containing nothing is also a subset of that. In fact, A set containing nothing would be a subset of every set. But not really. In reality, the creators of this had created another symbol that stands for "true" set: $subset$.

There are other kinds of sets as well, hence the... sub sets... for example, a set of nothing: $nothing$, or a universal set: $union$ (i lack a true universal set symbol, forgive me as i have to resort to alternate methods. ...They look close enough, though.) The universal set is especially interesting, capable of demonstrating or contraining the scope that the sets are operating at.
# Relationship between sets... (ominous foreshadowing)
You see, when sets are put together, one can create mini sets of data containg parts of the sets created. Putting two things next to eachother can create a new thing? How original!

Big boys out of the way first. The **Complements** of a set is the things ==not inside== said set, and i mean ALL of them by the way. **Difference** is more complicated, but is also less of a headache, simply pointing to the elements in the first set that is ==not inside== the second set.

Huh. So they are almost the same, then.

$sect$, or **Sect**, is the things inside the first set that is also ==inside== the second. While $union$, or **Union**, is simply everything ==inside== both sets.
